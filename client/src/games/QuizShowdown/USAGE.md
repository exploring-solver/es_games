# Science Quiz Showdown - Usage Guide

## Getting Started

### Main Menu

When you first launch Science Quiz Showdown, you'll see four game mode options:

1. **Single Player** - Practice solo and improve your skills
2. **Multiplayer** - Compete with up to 4 players
3. **Daily Challenge** - Complete today's special quiz
4. **Tournament** - Enter a bracket-style competition

## Single Player Mode

### Setup Phase

1. **Select Categories**
   - Click on category cards to select/deselect
   - Choose one or multiple categories
   - Click "Select All" for mixed questions
   - Each category is color-coded for easy identification

2. **Choose Difficulty**
   - **Mixed** - Questions of all difficulty levels (recommended)
   - **Easy** - Perfect for beginners
   - **Medium** - For intermediate knowledge
   - **Hard** - Advanced concepts
   - **Expert** - Maximum challenge

3. **Set Question Count**
   - Use the slider to select 5-50 questions
   - Shorter quizzes: Quick practice (5-15 questions)
   - Standard quizzes: Balanced challenge (15-25 questions)
   - Long quizzes: Marathon mode (25-50 questions)

4. **Start Quiz**
   - Click "Start Quiz" to begin
   - Timer starts automatically
   - First question appears immediately

### Playing the Quiz

#### Question Screen Elements

1. **Header Information**
   - Category badge with icon
   - Difficulty indicator (color-coded)
   - Point value
   - Timer bar (color changes as time runs out)

2. **Question Text**
   - Read carefully
   - Look for key words
   - Consider all options

3. **Answer Options**
   - Four multiple-choice options (A, B, C, D)
   - Click to select your answer
   - Selected option highlights
   - Cannot change after submission

4. **Progress Bar**
   - Shows current question number
   - Visual progress indicator
   - Questions remaining

5. **Game Statistics**
   - **Score** - Your current point total
   - **Streak** - Consecutive correct answers (with fire emoji)
   - **Accuracy** - Percentage of correct answers

#### Answering Questions

1. **Read the Question**
   - Take your time (but watch the timer!)
   - Understand what's being asked

2. **Select Your Answer**
   - Click on one of the four options
   - Answer submits automatically
   - No way to undo

3. **View Results**
   - ‚úì Correct answer highlighted in green
   - ‚úó Your wrong answer shown in red
   - Explanation appears below
   - Points awarded shown

4. **Continue**
   - Click "Next Question" button
   - New question appears
   - Timer resets

## Multiplayer Mode

### Setup

1. **Select Categories and Difficulty**
   - Same as single player
   - Settings apply to all players

2. **Player Ready**

   - Host starts when all ready

### Gameplay

1. **Same Question for All**
   - All players see same question
   - First to answer gets time bonus
   - Everyone can answer at their pace

2. **Live Leaderboard**
   - Real-time score updates
   - Current rankings shown

## Daily Challenge

### How It Works

1. **Same for Everyone**
   - All players get identical questions
   - Questions change daily at midnight
   - 20 questions, balanced difficulty

2. **Global Competition**
   - Compare scores worldwide
   - Daily leaderboard
   - Special badges for top performers

3. **One Attempt**
   - Single try per day
   - No retakes
   - Make it count!

### Tips for Daily Challenge

- Take your time (no pressure)
- Read explanations to learn
- Aim for high accuracy
- Build your daily streak

## Tournament Mode

### Structure

1. **Registration**
   - Up to 16 players
   - Seeded by skill level
   - Single elimination

2. **Rounds**
   - Round 1: All players compete
   - Round 2: Winners advance
   - Finals: Best of the best
   - Champion crowned

3. **Progression**
   - Win = Advance to next round
   - Lose = Eliminated
   - Difficulty increases each round

### Tournament Strategy

- **Early Rounds**: Build confidence with easier questions
- **Mid Rounds**: Use power-ups strategically
- **Finals**: Maximum difficulty, highest stakes
- **Consistency**: Accuracy matters more than speed

## Adaptive Difficulty System

### How It Works

The game learns your skill level:

1. **Initial Assessment**
   - Starts at medium difficulty
   - Observes your performance

2. **Continuous Adjustment**
   - Tracks correct/incorrect answers
   - Monitors response times
   - Adjusts question difficulty

3. **Category-Specific**
   - Different skill levels per category
   - Strong in physics? Get harder physics questions
   - Weak in chemistry? Get supportive chemistry questions

### Skill Progression

Track your improvement:

- **Novice** (0-30): Learning basics
- **Beginner** (30-45): Building foundation
- **Intermediate** (45-60): Solid understanding
- **Advanced** (60-75): Strong knowledge
- **Expert** (75-90): Near mastery
- **Genius** (90-100): Elite level

### Benefits

- Always challenging, never frustrating
- Optimal learning curve
- Personalized experience
- Track category mastery

- Some repetition normal
- Daily challenges always new
